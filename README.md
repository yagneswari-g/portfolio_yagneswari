# Advanced React Portfolio

This is a modern fresher-level portfolio built with React, Vite, Framer Motion and React Icons.

## Setup

```bash
npm install
npm run dev
```

Open the local URL shown in terminal, usually:

```bash
http://localhost:5173
```

## Replace Your Details

Open this file:

```txt
src/data/portfolioData.js
```

Update:

- Name
- Email
- Phone number
- GitHub link
- LinkedIn link
- Resume path
- Project GitHub links
- Live demo links
- Certificate links

## Resume

Put your resume PDF inside the `public` folder:

```txt
public/resume.pdf
```

Then the Download Resume button will work.

## Deploy to Netlify

```bash
npm run build
```

Then upload the `dist` folder to Netlify, or connect your GitHub repository to Netlify.
