#  React Portfolio

A modern and responsive developer portfolio built using **React, Vite, Framer Motion, and React Icons**.

## Features

* Responsive design
* Smooth animations
* Projects and skills sections
* Education and internship details
* Certificate links
* Contact information
* Resume download
* GitHub and live demo links

## Setup

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:5173
```

## Update Your Details

Edit:

```txt
src/data/portfolioData.js
```

Update your name, email, phone number, GitHub, LinkedIn, projects, certificates, and resume details.

## Resume

Place your resume inside:

```txt
public/resume.pdf
```

## Build

```bash
npm run build
```

## Deploy

Upload the generated `dist` folder to **Netlify**, or import the GitHub repository into **Vercel**.
